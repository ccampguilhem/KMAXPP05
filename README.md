## Évaluation (selon SGCE)
La note finale est calculée sur 4 notes ($n_i$) avec compensation par la 4ème épreuve ($n_4$).

**Coefficients :** $c_1=0.33, c_2=0.33, c_3=0.34, c_4=0$.

**Formule :**
$$\frac{\sum_{i=1}^{3} \max(n_i, n_4) \cdot c_i + n_4 \cdot c_4}{\sum_{i=1}^{4} c_i}$$
